关于Windows XP源码泄露

这次泄露

其实已经是旧闻了，这些源码很大一部分是从微软对高校的Windows Kernel Research项目来的，其他还有是真的泄露，如比尔盖茨的一些奇怪的私密视频文件，这些源码在4chan上首次公开，实际上已经在黑客私下传了很长时间，一个Youtuber叫NTDEV已经成功将其中的Windows XP源码编译成可以开机的系统，所以这些源码是真的😗

其实这次不知Windows XP，还有一下内容泄露，只不过完整度不高，泄露者提供的磁力连接中还有这些内容的源码：

  • Windows XP SP1
  • Windows 2000(基本不可以编译)
  • Windows Server 2003
  • Windows 10(极少)
  • Windows NT(两个版本)
  • Windows CE(几个版本)
  • MS-DOS(两个版本)
  • Xbox

国产系统？

当我逛知乎，偶然间让我发现迷幻的评论！

就…………分享下我对这些评论的观点？！

Microsoft® Windows®是闭源商业软件，微软拥有其版权，微软也拥有其专利，源码不可以商业使用，不然是违法行为，只可以用于学习用途，当时高校在研究Windows源码时，很多学生都说Windows源码写得很漂亮，很有参考意义

当然我也发现这些人的出发点是好的，打破美帝垄断，实际上Linux就已经可以做到了，但是国内软件开发商不争气，Linux的国内生态非常差，Windows反而到处都是软件，虽然Linux可以使用Wine运行，但又缩水：兼容性差

说到这里，我想到Wine通过参考这些源码提高系统兼容性哦🙂~yee

此外我怀疑到时候搞得不好就会出现一个基于Windows XP源码开发的国产系统（希望永远不会发生

此外我还发现有很多人希望源码集成到鸿蒙，开发鸿蒙PC版？！
