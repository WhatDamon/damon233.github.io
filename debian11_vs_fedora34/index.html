对比Debian 11与Fedora 34

最后更新于：1 年前

Start

难得玩一次Linux，并且还正式玩了一下Terminal，于是乎，就有了这玩意（感觉逻辑不太对

Middle

配置：

Macbook Pro 2011初，更换了SSD

其他配置按VMware默认来

系统：

Debian 11.2 Codename: bullseyes 64-Bits

Fedora 34 Workstation（Live CD）64-Bits

测试：

  1. 安装器

    主要依据安装程序的美观程度，易安装度等评价

    安装器 Debian Fedora
    GRUB美观 一般 配色好看，一般
    GRUB易用性 较低 较高
    安装UI 一般 一般
    试用 无 有
    安装速度 网络安装巨慢 快
    安装容易度 一般 友好
    安装方式 GUI+TUI GUI

  2. 系统引导

    GRUB引导器 Debian Fedora
    中文支持 有，但不完全 无
    壁纸 有，4:3 无
    易用性 OK OK
    系统引导器 Debian Fedora
    美观 差 好
    错误易处理 一般 较差

  3. 系统桌面

    桌面 Debian Fedora
    多桌面 支持 仅GNOME
    GNOME版本 默认3.38，更新麻烦 GNOME 40
    动画 无（可以开启 有，做得很好

  4. 应用

    应用 Debian Fedora
    格式 .deb .udeb .rpm
    兼容性 一般 较好
    终端 一般 一般
    应用商店 无法访问 有，但软件较少
    预装游戏 多 无
    预装应用量 很多 基础

  5. 可玩性

    可玩性 Debian Fedora
    sudoers 不设置不可用 默认可用
    su 可用 可用
    root账户 需设置 (UNKNOW)

  6. 稳定性

    稳定性 Debian Fedora
    错误概率 低 中

  7. 美观度

    美观 Debian Fedora
    活动 一般（GNOME3） 好看（GNOME40）
    主题 较少（需安装其他桌面） 1个（GNOME自带）

End

以上为个人测试总结，仅供参考

你可能会问deepin或者其他发行版呢？

我会回答：大多Linux发行版都是基于那俩系统开发（Fedora容易下载，Red Hat麻烦，这里就用了也是比较有代表性的Fedora）

例如Ubuntu、deepin等基于Debian，而Cent OS等基于Red Hat(这里是Fedora)，其实我建议都可以用用
