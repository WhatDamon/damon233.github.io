再见，Windows 8.1

最后更新于：7 个月前

告别！Windows 8.1

Windows 8.1将在2023年1月10日停止支持，无扩展安全更新计划，同时停止的还有Windows 7的3年扩展安全更新计划…

Windows 8.1的历史

Windows 8.1，开发代号为Windows Blue，内核从Windows 8的NT 6.2升级到了NT 6.3

2013年6月30日，微软在微软Build 2013大会中正式推出Windows 8.1预览版，到了2013年10月17日，微软发布Windows 8.1，并正式提供Windows 8.1的下载，所有安装了Windows 8的用户都可以通过Windows应用商店的更新中获得8.1版本更新（是的，应用商店，第一次，和当年的OS X一样，都后面和也苹果一样，应用商店和系统更新分开了）

到了2018年1月10日，微软正式宣布结束对Windows 8.1操作系统的主流支持，转入扩展支持阶段，这将持续到2023年1月10日

2022年，微软开始催促用户升级Windows 10或者最新的Windows 11

Windows 8.1的亮点

  1. 恢复了开始按钮，并且支持登陆以后直接进桌面
  2. 对于高分屏进一步优化
  3. Windows 8.1添加了新的邮件应用，同时Xbox Music也拥有了全新的版面设计
  4. 系统自带软键盘上增加了手势操作

Windows 8.1这一次的升级也是微软为了打造一款对于PC和平板都更加实用的平台这一尝试中最为成功的结果。开始按钮的回归表明微软意识到自己在Windows 8的计划当中走得有点太远了，这个小小的改动使得这个系统对于用户来说更加熟悉了。Windows 8.1还展示了微软想要统一旗下产品的决心和努力。桌面模式的灵活性依然是Windows 8.1当中最吸引人的使用方式。Windows 8用户当然都会欢迎Windows 8.1带来的改动，这些改动也应该清除某些部分的疑惑。

——搜狐网评

的确，甚至可以说，Windows 8是在为Windows 8.1铺路，就像Windows Vista给Windows 7铺路一样

相比同期的系统

Windows 8的Metro引领了计算机设计行业走向扁平化，虽然这本质上是对当时机能的一种妥协，Metro的灵动搭配毛玻璃太耗资源，但毫无争议，Windows 8无疑是做到了行业第一人，当年，OS X，iOS，以及大部分Linux发行版都在走拟物化的路线。相比Windows 7，Windows 8系列对UEFI有了质的飞跃，Metro风格的引导界面让人眼前一亮，甚至今天都还是那个样，Windows 8看中了平板的潜力，对触控做了许多优化，并且90年代的电脑无缘于Windows 8（不支持SSE2指令集），也让很多人用上了新电脑。微软似乎看中了云计算，SkyDrive（现OneDrive）等集成到系统。快速启动的引入，使Windows 8.1在相同设备开机速度比Windows 7快得多

相比于后面的系统

虽然Windows 8.1用机械硬盘也很慢，但相比Windows 10流畅太多太多，另外，Windows 10的Metro相比Windows 8.1也少了灵动，因为很多用户认为这几百毫秒完全没有意义。Windows 8.1的BUG相比Windows 10以及更新的Windows也少了很多，另外Windows 8.1的很多都得到了继承，还顺便解决了这个系统的很多痛点，甚至到了Windows 11，稍微修改下系统，就可以用上Charm栏，以及系统自带的Metro应用（Windows 11还原生支持Windows 8时代发布的Metro应用）

命运

虽然Windows 8的变化异常之大，改进特别多，但是只可惜，Windows 8以及Windows 8.1都是被遗忘的系统，就和Vista一样，生不逢时,过于超前，如果你问那些电脑小白，或许以为Windows 7的后代就是Windows 10，另外，大部分第三方软件都没有对Windows 8系列做专门的优化

尽管Windows 8.1也是一代被遗忘的系统，但是相比它的前代Windows 8而言，更受欢迎，因为很多用户痛点在这一代得到了解决

但是在Windows 11发布后，Windows 8.1再一次被网友所提起，因为这是最后一代稳定，流畅，且对老电脑友好的系统

升级吧！

Windows 8.1虽然带给你了很多惊喜，但是Windows 8.1不可能伴随你那么久，10年了，我们可以让Windows 8.1歇息了，同时也在保障你的设备安全，虽然第三方可以为它续命，但终究不是长久之计。让那些都走吧，坐和放宽，升级你的系统，向未来前进吧！

顺便一说

Google Chrome，Microsoft Edge已经开始对使用Windows 7、Windows 8以及Windows 8.1的用户发出警告，称这些系统将无法再接受新版本更新，要求用户赶快升级，如果您有暂时停留在Windows 8.1或者Windows 7的需求，可以试试Firefox，但是不久也会停止支持的！
