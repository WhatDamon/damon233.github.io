想用Virtual Box 7.0？我劝你再观望一下

最后更新于：2024-08-10T15:48:48+08:00

先说定论，不推荐！

现在，我需要告诉你Virtual Box 7.0相比6.0有哪些进步，哪些退步，以帮助你选择是否更新！

这次我不说废话，晒图自己看吧。。。

当然，如果你升级了，那么除了卸载重装VBox，都无法回到6.0

缺点

  1. 到了7.0，2.5k屏的显示效果大幅度缩水，字体于按钮得巨大

  2. 插图开始缝合，出现了扁平和渐变杂糅乱象

  3. 新建虚拟机UI大幅缩水，看起来更加粗糙

  4. 中文翻译不全，严重影响体验

  5. 虚拟机启动效率变慢，每次启动你都要等最少15秒

  6. 设计上颜色总体偏灰，感觉看起来更脏

优点

  1. 暗色模式（终于有了！！！！！！

  2. 虚拟机TPM配置（谢谢你啊

  3. DirectX 11的3D加速
