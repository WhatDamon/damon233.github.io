你应该用什么启动器？

最后更新于：16 天前

最有保障

Minecraft Launcher(官方启动器)

虽然有一点要承认，官方启动器在国内经常发抽，但作为官方启动器，可以说最有保障了，并且对小白比较友好

优点
  1. 可以实时追踪MC新闻
  2. 可以在启动游戏时自动下载Java
  3. 源有保证，并且官方出品，安全系数最高
  4. 可以启动地下城和基岩版（后期还有可能可以启动传奇）
  5. 跨平台（其他平台只可以玩Java版）
  6. 还可以体验试玩版的MC（顺便一说，试玩模式下旧版MC是玩完整版）
  7. 美观
  8. 方便管理账号皮肤和披风
  9. 可以搞XGP等
缺点
  1. 对于中国地区下载速度过慢
  2. 体积过大
  3. 经常发抽
  4. Mods管理不方便
  5. 不支持启动器内安装Forge，Fabric，Optifine，Liteloader，Quilt
  6. 有时会出现特别阴间的BUG
  7. 不买就玩不了完整版

小白必备

Plain Craft Launcher 2(PCL2)

非常适合小白使用，配置很无脑（注：不是PCL1启动器）

优点
  1. 界面丝滑
  2. 支持下载只在Wiki出现或者单独发布的版本
  3. Peer to Peer（P2P）联机
  4. 下载游戏速度快（视网络环境而定）
  5. 可以自定义主页（前提你要会XAML）
  6. 可以直接从CurseForge下载Mod
  7. 配置轻松
  8. 可以自动分类游戏
  9. 可以轻松在各.minecraft游戏文件夹切换
缺点
  1. 闭源，无法通过改源码实现自己想要的功能
  2. 不支持从Modrinth下载Mod
  3. 自定义性较差
  4. 只支持Windows
  5. 体验最新功能要花钱买

BakaXL

作为一个很奇葩的启动器，这个启动器有很多有意思的功能

优点
  1. 核心开源，方便自制启动器，或者让软件更怪
  2. 配置简单
  3. 华丽
  4. 支持Quilt配置
  5. 完全免费
  6. 自定义性强
  7. 多语言
缺点
  1. 占用偏高
  2. 总体风格不适合新版Windows 11
  3. 只支持Windows
  4. 升级强制性

折腾性

Hello Minecraft Launcher(HMCL)

作为一直在维护的老国产MC启动器，可以说十分经典了、

优点
  1. 完全免费开源
  2. 支持离线联机
  3. 跨平台
  4. 皮肤管理方便
  5. 报错信息详细
  6. 自定义性极高
缺点
  1. 并非完全无脑
  2. Optifine和Fabric默认无法安装在一起
  3. 界面较为传统，不新奇

MultiMC

国外的优秀启动器，在中国也非常有名

优点
  1. 自定义性强
  2. 免费开源（开发者不希望第三方编译）
  3. 版本管理简单
  4. 支持很多其他启动器没有功能
缺点
  1. 必须要登陆Mojang或Microsoft账号才可以玩
  2. 体积稍微优点大
  3. 多文件，不方便转移

WARNING!

以下启动器千万别使用，他们都有风险！！

  1. PolyMC

    这是MultiMC的分支，本身很优秀，但是其密钥库发生泄露，作者解散了开发团队，软件已经变得不可信任
